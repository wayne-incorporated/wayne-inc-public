웨인OS

크롬 OS의 오픈 소스 구현

서버용 리눅스, PC용 윈도우, 웹 씬 클라이언트용 웨인OS. 웹 전용 PC, 학교/공공용 PC에서 저가형 산업용 하드웨어에 이르기까지 유일한 웹 씬 클라이언트 OS 솔루션입니다.


위치 설정

크로미움OS는 2009년부터 구글에서 시작한 차세대 OS 프로젝트입니다. (주)웨인은 품질을 향상시키고 씬 클라이언트 디바이스에 맞춘 업계 요구 사항을 통합합니다.

크롬OS와 웨인OS는 모두 크로미움OS에서 파생되었지만, 웨인 OS는 향상된 비즈니스 유연성을 제공하여 차별화됩니다. 웨인OS는 사용자가 소스 코드와 바이너리를 수정, 복사, 배포 등 상업적 목적으로 활용할 수 있게 해줍니다.

중개 제품으로 자리매김한 웨인OS는 씬 클라이언트 산업 내에서 최종 제품 및 서비스 생산에 활용되고 있습니다.


크로미움OS의 한계

  • 제한 사항

사용자 앱/코드를 제한하고 보안, 성능, 관리를 위해 웹 전용 액세스를 허용하는 것이 크로미움OS의 철학입니다. 따라서 기존 OS와 같은 다목적/애플리케이션에는 적합하지 않습니다.

  • 개발

오픈 소스 크로미움OS의 공식 바이너리/배포 버전이 없기 때문에 사용자는 소스 코드를 직접 수정하고 빌드해야 합니다. 커널 수정, 기능 추가 및 라이선스 검사에는 고급 개발 기술과 시간이 필요합니다.

  • HW 호환성

크로미움OS는 일반 PC 하드웨어와 잘 호환되지 않습니다. 크로미움OS의 재개발 버전은 더 나은 호환성을 가지고 있지만 여전히 불완전합니다. 따라서 배포 전 테스트가 필요합니다. 문제는 주로 여러 GPU 무선 LAN 카드와 I/O 장치로 인해 발생하기 때문입니다.


웨인 OS의 장점

  • 사전 제작 및 사용자 지정 가능

크로미움OS를 개발하려면 상당한 인적 자원이 필요합니다. (주)웨인이 무료로 개발 및 배포함에 따라 이제 웨인OS를 쉽게 사용자 지정하고 자체 브랜드 OS를 출시할 수 있습니다.

  • 무결성

사용자 수준에서 무결성을 제공합니다. 앱/바이러스를 설치/실행할 수 없으며, 사용자가 웹을 사용하도록 강제할 수 없습니다.

  • 경량

속도 저하가 없습니다. 저가형 하드웨어를 위해 설계되었습니다.

  • TCO(총 소유 비용)

기기의 생산/유지보수/교육에 필요한 TCO를 절약하세요. 무제한으로 여러 기기에 OS를 설치할 수 있습니다.
